# Stock Trading Platform

This project is developed as part of the Java Programming Internship at CodeAlpha.

## Description
A console-based Java application that simulates a basic stock trading platform.
Users can view stocks, buy and sell shares, and track portfolio balance.

## Technologies Used
- Java
- OOP Concepts
- ArrayList
- Console I/O

## How to Run
javac Stock.java Transaction.java User.java StockTradingPlatform.java  
java StockTradingPlatform
